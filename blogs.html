The basic difference between var , let and const is that :

- The scope of the var keyword is the global or function scope. It means variables defined outside the function can be accessed globally, and variables defined inside a particular function can be accessed within the function. User can re-declare variable using var and user can update var variable.

- The scope of a let variable is only block scoped. It canâ€™t be accessible outside the particular block ({block}).

- The const keyword has all the properties that are the same as the let keyword, except the user cannot update it.
